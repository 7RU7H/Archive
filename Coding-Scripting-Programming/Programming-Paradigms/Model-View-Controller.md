# Model View Controller

[Model-View-Controller](https://en.wikipedia.org/wiki/Model%E2%80%93view%E2%80%93controller) is a design pattern 
- MVC:
	- `User` - uses `Controller`
	- `Controller` manipulates -> `Model`
	-  `Model` updates -> `View`
	- `View` sees `User`
