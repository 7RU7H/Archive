# MISP

[MISP (Malware Information Sharing Platform)](https://www.misp-project.org/) is a threat information sharing platform that is free & open source software. It collects, normalises, correlates and enriches data and then updates defensive and analyst tools with the output.

- [Github - MISP](https://github.com/MISP/)
- [MISP -Documentation](https://www.misp-project.org/documentation/)
