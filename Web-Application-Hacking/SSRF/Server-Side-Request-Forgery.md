# Server-Side Request Forgery

Server-Side Request Forgery (SSRF)

- Types
	- Reflected
	- Blind
- Found:
	- Full URL used in a parameter in address bar
	- Hidden field in a form
	- Partial URL as just a hostname
	- Path of a URL 

## Mitigation

- Deny List
- Allow List
- Open redirect
