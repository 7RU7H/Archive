# Prompt-Engineering

Prompt Engineering is the act of inventively trying to engineer communication to be used for or against a target AI agent, [[LLMs]] or `xyz`.

Components of a Prompt:
- Instruction
- Context
- Output Format
- Constraints

Positive traits:
- Specificity 


| |**System Prompt**|**User Prompt**|
|---|---|---|
|**Set by**|Developer / application|End user|
|**Nature**|Immutable, constant|Dynamic, session-specific|
|**Purpose**|Establishes identity, rules, and safety boundaries|Carries task-specific requests and data|
|**Example**|"Never execute code. Always be helpful and professional."|"Summarise this document for me."|
|**Priority**|High-priority context that shapes overall behaviour|Acted on within the system prompt's constraint|
#### Advanced Prompting Techniques

Shot Spectrum
- Give N number of examples

Chain of Thought
- Tasks the model to break down complex tasks into intermediate steps
