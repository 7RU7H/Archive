# CAPA

CAPA (Common Analysis Platform for Artefacts) is a tool developed by the FireEye Mandiant team.

```powershell
capa.exe $bad.bin
capa.exe -h
capa.exe -v # -vv for more verbosity
capa.exe $bad.bin -j # output to json file
```

https://mandiant.github.io/capa/explorer/#/ *"capa Explorer Web is a web-based tool to explore the capabilities identified by capa. This tool allows you to interactively browse and display capa results in multiple viewing modes."*
