# neo4j


## Startup:

### Per shell session start with
```bash
ulimit -n 40000 # increase history
# THEN
sudo neo4j console
# THEN goto localhost:7474
```
Default credentials
USER: neo4j
PASS: neo4j

[Change Password](https://neo4j.com/docs/operations-manual/current/configuration/password-and-user-recovery/)
